Q1. Which of the following is the correct chronological order of UI evolution?
- A. GUI → CLI → Web → Mobile
- B. CLI → GUI → Web → Mobile
- C. Web → CLI → GUI → Mobile
- D. Mobile → Web → GUI → CLI

<details>
<summary>Show Answer</summary>

**Answer:** B  
**Explanation:** Command-line interfaces came first, then graphical user interfaces, followed by web applications, and finally mobile apps.

</details>
